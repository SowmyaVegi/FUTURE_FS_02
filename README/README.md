# Mini CRM

A Client Lead Management System built using:

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB

## Features

✔ Add Leads

✔ View Leads

✔ Update Lead Status

✔ Delete Leads

✔ Notes Management

✔ Responsive Design

## Installation

### Backend

npm install

npm run dev

### MongoDB

Start MongoDB Service

### Frontend

Open index.html

## API Endpoints

GET /api/leads

POST /api/leads

PUT /api/leads/:id

DELETE /api/leads/:id
